# `地图lite` 小程序

> **我没有开源这个小程序的代码，所以网络上看到的肯定不是我的代码咯，很囧。**

一个地图小程序，主要提供了以下的功能：

- 驾车、公交、步行、骑行的路线规划
- 公交地跌换乘方案
- 天气预报
- 地点信息及周边信息查询
